Plantilla de un documento HTML para empezar a desarrollar una pagina.

```html

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>

	<header>
	</header>
	
	<main>
	</main>
	
	<footer>
	</footer>
    
</body>
</html>

```
